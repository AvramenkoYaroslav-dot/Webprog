Мої гіперпосилання

Список корисних ресурсів Інтернету

Поштова служба Gmail
Gmail — популярна безкоштовна електронна поштова служба від компанії Google, відома своїм зручним інтерфейсом та інтеграцією з іншими сервісами.
Пошуковий сервер Google
Google — один із найпопулярніших у світі пошукових серверів, який надає доступ до величезної кількості інформації в Інтернеті.
Українська Вікіпедія
Вільна енциклопедія, створена спільнотою користувачів, що містить мільйони статей українською мовою на різні теми.
MDN Web Docs
Офіційна документація для веб-технологій (HTML, CSS, JavaScript), незамінний ресурс для кожного веб-розробника.
GitHub
Найбільша веб-служба для хостингу ІТ-проектів та їх спільної розробки, що використовує систему контролю версій Git.
Відеохостинг YouTube
Найбільший у світі відеохостинг, де користувачі можуть завантажувати, переглядати та ділитися відеоконтентом.
Професійна мережа LinkedIn
Соціальна мережа, орієнтована на пошук та встановлення ділових контактів, а також на професійний розвиток.
